JavaScript for Python Developers

Since you know Python, here's how JavaScript compares:

Python
# Function definition
def greet(name):
    print(f"Hello, {name}!")

# Function call
greet("Trix")

# Variable assignment
counter = 0

# Conditionals
if counter > 5:
    print("Greater than 5")
else:
    print("5 or less")
JavaScript
// Function definition
function greet(name) {
    console.log(`Hello, ${name}!`);
}

// Function call
greet("Trix");

// Variable assignment
let counter = 0;

// Conditionals
if (counter > 5) {
    console.log("Greater than 5");
} else {
    console.log("5 or less");
}
